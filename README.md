# Fall Jens Söring – Ermittlerboard

Statisches Ermittlerboard zu den Haysom-Morden (Bedford County, Virginia, 30. März 1985): Beziehungsnetz der Beteiligten, Zeitstrahl 1984–heute, Quellen-Ampel — dazu die Recherche-Werkstatt [mitmachen.html](https://bedford-1985.github.io/mitmachen.html).

**Live:** https://bedford-1985.github.io/

Die Seite bezieht keine Position zur Schuldfrage; es gilt die rechtskräftige Verurteilung von 1990, Sörings Unschuldsbeteuerung und die anhängigen Verfahren sind dokumentiert.
