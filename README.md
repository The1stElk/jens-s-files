# Fall Jens Söring – Ermittlerboard

Statisches Ermittlerboard zu den Haysom-Morden (Bedford County, Virginia, 30. März 1985): Beziehungsnetz der Beteiligten, Zeitstrahl 1984–heute — dazu der Reiter „Dokumente & Quellen“ mit kuratierten, ampel-eingestuften Fundstellen (nur Links, keine gespiegelten Dokumente). Die frühere Recherche-Werkstatt (mitmachen.html) ist eingestellt; die Adresse leitet auf den Quellen-Reiter um.

**Live:** https://bedford-1985.github.io/

Die Seite bezieht keine Position zur Schuldfrage; es gilt die rechtskräftige Verurteilung von 1990, Sörings Unschuldsbeteuerung und die anhängigen Verfahren sind dokumentiert.
